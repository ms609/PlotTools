# Changelog

## PlotTools v0.4.0 (2025-01-29)

- Add colour-blind friendly palettes `cbPalette12`, `cbPalette15`,
  `cbPalette24` and deuteranopic equivalents `altPaletteXX`.

## PlotTools v0.3.1 (2024-08-27)

CRAN release: 2024-09-03

- [`SpectrumLegend()`](https://ms609.github.io/PlotTools/reference/SpectrumLegend.md)
  supports log scales.

## PlotTools v0.3.0 (2023-10-30)

CRAN release: 2023-10-30

- [`Col2Hex()`](https://ms609.github.io/PlotTools/reference/Col2Hex.md)
  converts R colour names to hexadecimal codes.

## PlotTools v0.2.1 (2023-09-28)

CRAN release: 2023-09-28

- Default to `SpectrumLegend(lend = "butt")` to avoid artefacts with
  semi-transparency
  ([\#3](https://github.com/ms609/PlotTools/issues/3)).
- Use more intuitive argument order in
  [`SpectrumLegend()`](https://ms609.github.io/PlotTools/reference/SpectrumLegend.md).

## PlotTools v0.2.0 (2023-05-19)

CRAN release: 2023-05-19

- Add polygon functions, initially from “Ternary”

## PlotTools v0.1.0 (2023-04-18)

CRAN release: 2023-04-18

- Initial CRAN release, replacing previous implementation in package
  “TreeTools”.
- Functions
  [`SizeLegend()`](https://ms609.github.io/PlotTools/reference/SpectrumLegend.md)
  and
  [`SpectrumLegend()`](https://ms609.github.io/PlotTools/reference/SpectrumLegend.md)
  annotate continuous plotting variables.
